**BitBattles** — это экспериментальный автобаттлер, в котором интеллект юнитов и тактика противника создаются с помощью **гибридного подхода**: обучение с подкреплением (Deep Q-Network) управляет поведением юнитов игрока, а генетический алгоритм эволюционирует расстановку врагов.

## Ключевые особенности

| Компонент | Технология | Результат |
|-----------|------------|-----------|
| **Поведение юнитов игрока** | DQN (Deep Q-Network) | Юниты учатся атаковать, укрываться и лечиться |
| **Тактика врага** | Генетический алгоритм (нейроэволюция) | Враг эволюционирует, находя оптимальные расстановки |

| **Визуализация обучения** | Jupyter Notebook + Matplotlib | Графики Loss, Win Rate, Fitness, Diversity |

## Как это работает

### RL-агент (управление юнитами игрока)
- **Алгоритм**: DQN с целевой сетью и Replay Buffer (50 000 записей).
- **Нейросеть**: 27 → 128 → 128 → 64 → 5 (выход: 5 действий: вверх/вниз/влево/вправо/стоять).
- **Входные данные**: координаты, HP, уровень, тип, ближайший враг, 8 соседей, 9 клеток в радиусе атаки, укрытие за камнем.
- **Система наград** (фрагмент):
  - Убийство врага: `+5`
  - Подбор хилки: `+3`
  - Укрытие за камнем: `+0.5`
  - Смерть: `-10`
- **Результат**: Win Rate **~52%**, нейросеть стабильно принимает решения.

### GA-агент (расстановка врагов)
- **Алгоритм**: генетический алгоритм + нейросеть (нейроэволюция).
- **Нейросеть**: 102 → 64 → 32 → 225 (9 типов юнитов × 25 клеток, Sigmoid).
- **Эволюция**:
  - Популяция: 15 особей
  - Турнирная селекция (размер 4)
  - Равномерный кроссовер
  - Гауссовская мутация (адаптивная)
  - Элитизм (топ-10%)
- **Fitness-функция** учитывает: убийства, урон, выживших, полную победу.
- **Результат**: Максимальный Fitness **143.0**, Win Rate до **86.7%**.

### Сетевое взаимодействие
- **RL-сервер**: порт `65432`, JSON (`{state, reward, done, unit_id}` → `{action}`).
- **GA-сервер**: порт `5005`, строковый формат `бюджет|фитнес|состояние_поля` → `тип:уровень,...`.

## Геймплей и механики

### Игровое поле
- Сетка **5×10**: 5 нижних рядов — игрок, 5 верхних — враг.
- Препятствия (камни) и лечащие объекты (хилки) генерируются случайно.

### Юниты

| Тип | HP | Урон | Радиус атаки | Стоимость |
|-----|----|------|--------------|------------|
| Рыцарь (Knight) | 15 | 5 | 1 | 3 |
| Лучник (Archer) | 8 | 4 | 3 | 2 |
| Маг (Mage) | 6 | 7 | 4 | 4 |

### Система уровней
- **Слияние**: два одинаковых юнита → улучшенный (уровень 2, затем 3).
- Характеристики умножаются: ×2 для 2 уровня, ×3.5 для 3 уровня.
- Меняется спрайт, добавляются звёзды.

### Экономика
- Начальный эликсир: **10 монет**.
- Пополнение между раундами: **+4 монеты**.
- Драфт: случайные 3 карты (можно купить или обновить за 1 монету).
- Продажа юнита: возврат стоимости −1.

### Бой
- **Пошаговый**: все юниты одновременно выбирают цель, двигаются, атакуют.
- Атака автоматическая при входе в радиус.
- Юниты могут **укрываться за камнями** (дальнобойные).
- Лечение при подборе аптечки (**+50% HP**).

### Раунды
- **Best of 9** (до 5 побед).
- Выжившие юниты игрока **сохраняются** между раундами.
- Вражеская расстановка генерируется **GA-сервером** заново каждый раунд.
- Таймер **20 секунд** на расстановку (автостарт).

## Установка и запуск

### Требования
- Windows 10/11 или macOS Tahoe (для сервера)
- Unity 6000.0.25f1 (LTS)
- Python 3.10+ с пакетами:
  ```bash
  pip install torch numpy matplotlib notebook    Разрешение экрана: 1920×1080 (рекомендуется)

Шаги

    Клонировать репозиторий
    bash

    git clone https://github.com/Varied-Kratos/BitBattles.git
    cd BitBattles

    Запустить RL-сервер (в отдельном терминале)
    bash

    cd AI/RL_Server
    python rl_server.py   # порт 65432

    Запустить GA-сервер (в другом терминале)
    bash

    cd AI/GA_Server
    python ga_server.py   # порт 5005

    Открыть проект в Unity 6

        File → Open Project → выбрать папку BitBattles

        Дождаться импорта ассетов

    Запустить сцену

        Assets/Scenes/MainMenu.unity

        Нажать Play

    Если сервера не запущены, игра работает с ИИ по умолчанию (случайные действия / стандартная расстановка).
